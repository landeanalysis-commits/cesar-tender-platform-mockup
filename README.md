# Cesar International Tender Platform - Static Mockup

A static HTML mockup demonstrating the visual design and user flows for an international tender/task marketplace platform.

## Overview

This mockup showcases:
- Customer journey (posting tasks, reviewing offers)
- Provider journey (browsing tasks, submitting offers)
- Clean Scandinavian minimalist design
- Mobile-responsive layout

## Pages Included

### Public Pages
- Landing page
- Login
- Registration

### Customer Portal
- Dashboard
- Post Task
- My Tasks
- View Offers

### Provider Portal
- Dashboard
- Browse Tasks
- Task Detail & Offer
- Profile

### Shared
- Messages

## Technology

- Pure HTML5
- Tailwind CSS (via CDN)
- No backend functionality (static mockup only)
